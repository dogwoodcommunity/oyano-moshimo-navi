# 家族3組テスト 当日シート

目的は「便利かどうか」を聞くことではなく、家族が自力で使い始められるか、必要な時に戻ってこられるか、お金を払う理由があるかを確認すること。

## テスト前に伝えること

- これは正式販売前の確認です。
- 親御さんの入院、介護、認知症、死亡などの情報は、必要な範囲だけ入力してください。
- 暗証番号、パスワード、マイナンバー画像は入力しないでください。
- 法律、税金、医療の判断は専門家に確認してください。このサービスは判断を断定しません。
- 途中でやめても大丈夫です。削除希望があれば受け付けます。

## その場で見たい行動

1. Web入口を開く
   - URL: `https://oyano-moshimo-navi.vercel.app`
   - 見ること: 入口が分かるか。説明なしで「まず状況を選ぶ」へ進めるか。

2. 状況整理を完了する
   - 見ること: 5分以内に最後まで行けるか。
   - 詰まった言葉、迷った設問、戻った箇所をメモする。

3. 結果画面を見る
   - 見ること: 「まずやること3つ」「期限のあるタスク」が理解されるか。
   - 聞くこと: 「この中で、今日やるならどれですか？」

4. アプリに保存する
   - 見ること: 結果画面からアプリ保存の意味が分かるか。
   - 見ること: Magic Linkで戻って、家族ボードに対象者とタスクが出るか。

5. 家族ボードを見る
   - 見ること: 「今日・期限超過」「7日以内」「担当未定」が分かるか。
   - 聞くこと: 「誰がやるか決まっていないものは見つけやすいですか？」

6. 担当者を変える
   - 見ること: タスクの担当チップを押して、自分または家族に割り当てられるか。

7. 通知設定を見る
   - 見ること: 通知が毎日使わせるものではなく、期限用だと伝わるか。
   - 見ること: ON/OFFが保存されるか。

8. 家族招待を試す
   - 見ること: 招待リンクを作って共有できるか。
   - 無料枠: オーナー以外2名まで。

## 7日後に聞くこと

- 7日間で、自分からもう一度開きましたか。
- 開いた場合、何を確認しましたか。
- 開かなかった場合、何があれば戻りましたか。
- 通知が来たら開きたいと思いますか。

## 支払意思の聞き方

「便利ですか？」とは聞かない。

代わりに、金額を出して聞く。

- 入力内容を人が確認し、家族会議用の整理レポートが出て、相談先候補まで整理されるなら、9,800円(税込)で申し込みますか。
- 申し込まない場合、いくらなら検討しますか。
- 何が足りないと払えないですか。

## 記録する数字

- 自力でWeb整理完了: 3組中何組
- アプリ保存完了: 3組中何組
- 家族招待を1人以上送れた: 3組中何組
- 7日後に自発的に再訪: 3組中何組
- 9,800円の支払意思あり: 3組中何組
